# Vibe Coding 最佳实践：文档先行

vibe coding经验：在让AI写代码前，先写文档栈-项目知识库（如PRD、技术栈说明等）+持久层文件（如CLAUDE.md声明AI每次会话必须遵循的规则约束、大型项目的进度文件等）。AI做的事情是生成样板代码、写重复逻辑、快速探索方法；那我的话应该是掌握项目的核心概念、阅读AI的代码、发现哪里错了等，不能依赖AI做一切，类似在流沙上构建项目。好的文档设计是帮助自己和AI在未来理解项目的关键，README解释项目是什么、CLAUDE.md声明规范是什么、以及经验文档说明哪里需要注意什么。
